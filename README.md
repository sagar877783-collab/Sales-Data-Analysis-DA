Sales Data Analysis project using Python, Pandas and Matplotlib
# Sales Data Analysis Project

## Objective
Analyze sales data using Python to identify trends and business insights.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Monthly Sales Analysis
- Product-wise Sales Analysis
- Region-wise Sales Analysis
- Data Visualization

## Run Project

```bash
pip install -r requirements.txt
python main.py
```

## Output
- Charts and Graphs
- Sales Reports
- Trend Analysis
